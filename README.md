# lost-found-information-system


Lost & Found Web Application

This is a simple Lost & Found web application developed using Python and Flask.
Users can register, login, and post lost or found items with description and contact number.

Project Features:

User Signup and Login

Gmail Email Validation

Add Lost Item

Add Found Item

Contact Number and Description

Admin Panel

SQLite Database

Technologies Used:

Python

Flask

SQLite

HTML

CSS

Project Structure:

app.py
database.db
templates folder
static folder

How to Run:

Install Python

Install Flask using: pip install flask

Run the project using: python app.py

Open browser and go to: http://127.0.0.1:5000
